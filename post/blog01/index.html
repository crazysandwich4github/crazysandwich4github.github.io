Creat a new blog

Hugo

  1. hugo new post/blog01.md

  2. vi blog01.md

  3. hugo –theme=dream –baseUrl=“https://crazysandwich4github.github.io/" –buildDrafts

Git

  1. git status . -s

  2. git add .

  3. git commit -m “creat a new blog”

  4. git push

Publish blog

Github

  1. Creat a new branch named YOURGITHUBNAME.github.io

Git

  1. git remote add origin https://github.com/YOURGITHUBNAME/YOURGITHUBNAME.github.io.git

  2. git push -u origin master
